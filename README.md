### Spotify Popularity Analysis: Machine Learning & Predictive Modeling  

This project leverages machine learning to analyze and predict song popularity on Spotify using audio features like energy, danceability, and loudness. It implements a comprehensive suite of algorithms to uncover patterns and estimate track success.

### Key Features 


Supervised Learning: Implements Logistic Regression, Naive Bayes, Decision Trees, Random Forests, KNN, XGBoost, and both Linear and Polynomial Regression for classification and popularity estimation.


Unsupervised Learning: Applies K-Means Clustering and DBSCAN to identify hidden patterns and natural groupings within the music data.


Performance Metrics: Evaluates models using Accuracy, Precision, Recall, and F1-score to ensure predictive reliability.


Data Visualization: Interprets clustering results to gain deep insights into the dataset's underlying structure.

### Tech Stack 


Language: Python 


Algorithms: Supervised (Classification/Regression) and Unsupervised (Clustering) 


Evaluation: Multi-metric performance analysis and data visualization
